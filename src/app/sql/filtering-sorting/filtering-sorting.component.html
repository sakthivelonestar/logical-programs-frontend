What: Array methods in JavaScript are built-in functions that can be used on arrays to manipulate, transform, and extract data from them.

Why: Array methods are important because they provide efficient ways to work with collections of data, reduce code complexity, and enable functional programming patterns.

Where: Array methods are used throughout JavaScript applications for data processing, filtering, sorting, transformation operations, and whenever working with collections of items.

Step 3: Sorting and Limiting Results

-- Sort by single column
SELECT * FROM employees ORDER BY salary DESC;

-- Sort by multiple columns
SELECT * FROM employees ORDER BY department ASC, salary DESC;

-- Limit number of results
SELECT * FROM employees ORDER BY salary DESC LIMIT 10;

-- Pagination with LIMIT and OFFSET
SELECT * FROM employees ORDER BY employee_id LIMIT 10 OFFSET 20;

-- Top earners in each department
SELECT department, first_name, last_name, salary
FROM employees 
WHERE salary > 50000
ORDER BY department, salary DESC;

Sorting, limiting, and pagination techniques for better data presentation and performance.
