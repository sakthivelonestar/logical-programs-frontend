What: Array methods in JavaScript are built-in functions that can be used on arrays to manipulate, transform, and extract data from them.

Why: Array methods are important because they provide efficient ways to work with collections of data, reduce code complexity, and enable functional programming patterns.

Where: Array methods are used throughout JavaScript applications for data processing, filtering, sorting, transformation operations, and whenever working with collections of items.

Step 4: Aggregate Functions and Grouping

-- Basic aggregate functions
SELECT COUNT(*) AS total_employees FROM employees;
SELECT AVG(salary) AS average_salary FROM employees;
SELECT MAX(salary) AS highest_salary FROM employees;
SELECT MIN(salary) AS lowest_salary FROM employees;
SELECT SUM(salary) AS total_payroll FROM employees;

-- Grouping data
SELECT department, COUNT(*) AS employee_count
FROM employees 
GROUP BY department;

-- Grouping with conditions
SELECT department, AVG(salary) AS avg_salary
FROM employees 
GROUP BY department
HAVING AVG(salary) > 45000;

-- Multiple aggregations
SELECT 
    department,
    COUNT(*) AS employee_count,
    AVG(salary) AS avg_salary,
    MAX(salary) AS max_salary,
    MIN(salary) AS min_salary
FROM employees 
GROUP BY department;
