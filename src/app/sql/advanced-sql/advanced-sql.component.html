Interview Questions

1) What is Indexing? Create Syntax and Drop Syntax

Answer: An index is a database structure that improves the speed of data retrieval operations on a table. Think of it like an index in a book - it helps you quickly find specific information without scanning every page.
Create Index Syntax:
CREATE INDEX index_name ON table_name(column_name);
Drop Index Syntax:
DROP INDEX index_name ON table_name;

2) What are Constraints? Syntax and How to Drop

Answer:Constraints are rules applied to database tables to ensure data integrity and enforce business rules. They prevent invalid data from being inserted into the database. Common constraints include NOT NULL, UNIQUE, PRIMARY KEY, FOREIGN KEY, CHECK, and DEFAULT.
Syntax to Add Constraint (Example):
ALTER TABLE table_name ADD CONSTRAINT constraint_name CHECK (column_name > 0);
Syntax to Drop Constraint (MySQL Example):
ALTER TABLE table_name DROP CONSTRAINT constraint_name;
Note: In MySQL, for foreign keys you use DROP FOREIGN KEY.
