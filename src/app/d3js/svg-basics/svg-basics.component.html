What: In D3.js, SVG (Scalable Vector Graphics) is the core technology used to draw shapes like circle, rect, line, and path in the browser. D3 manipulates these SVG elements dynamically.

Why: SVG is resolution-independent, interactive, and integrates seamlessly with D3 for data-driven visualizations. It allows precise control of shapes, colors, and positions.

Where: Use SVG in D3.js whenever you need to create charts, graphs, maps, or custom visualizations that require scalable, high-quality rendering directly in the DOM.

Drawing Shapes with SVG in D3.js


<!-- ================= Circle ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- Circle Example -->
  <!-- 
    cx = center x-axis
    cy = center y-axis
    r  = radius
  -->
  <circle cx="80" cy="100" r="50"fill="lightblue" stroke="black" stroke-width="2"></circle>
</svg>


<!-- ================= Rectangle ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- Rectangle Example -->
  <!-- 
    x = start position (x-axis)
    y = start position (y-axis)
    width = rectangle width
    height = rectangle height
  -->
  <rect x="150" y="50" width="100" height="80"fill="orange" stroke="black"></rect>
</svg>

<!-- ================= Line ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- Line Example -->
  <!-- 
    x1,y1 = starting point
    x2,y2 = ending point
  -->
  <line x1="20" y1="180" x2="280" y2="180" stroke="green" stroke-width="3"></line>
</svg>

<!-- ================= Ellipse ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- Ellipse Example -->
  <!-- 
    cx = center x
    cy = center y
    rx = radius on x-axis
    ry = radius on y-axis
  -->
  <ellipse cx="100" cy="80" rx="50" ry="30" fill="lightgreen" stroke="black"></ellipse>
</svg>

<!-- ================= Polygon (Triangle) ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- Triangle Example (Polygon) -->
  <!-- 
    points = list of x,y coordinates
    (150,20)  = top point
    (50,180)  = bottom-left
    (250,180) = bottom-right
  -->
  <polygon points="150,20 50,180 250,180" fill="yellow" stroke="black" stroke-width="2"/>
</svg>

<!-- ================= Path (Diagonal Line) ================= -->
<svg width="300" height="200" style="border:1px solid black;">  
 <path d="M20 180 L280 20" stroke="blue" stroke-width="2" fill="none" />
</svg>

<!-- ================= Path (Open Shape: H & V) ================= -->
<svg width="300" height="200" style="border:1px solid black;">  
 <path d="M10 10 H290 V190 H250" stroke="green" stroke-width="2" fill="none" />
</svg>

<!-- ================= Path (Closed Shape with Z) ================= -->
<svg width="300" height="200" style="border:1px solid black;">  
 <path d="M10 10 H290 V190 H250 Z" stroke="green" stroke-width="2" fill="none" />
</svg>

<!-- ================= Path (Quadratic Curve Q) ================= -->
<svg width="300" height="200" style="border:1px solid black;">
  <!-- The path starts at M20 120 -->
  <!-- It then draws a curve toward the control point Q150 0 -->
  <!-- And ends at the final point 280 120 -->
  <path d="M20 120 Q150 0 280 120" stroke="black" fill="transparent" stroke-width="2" />

</svg>
