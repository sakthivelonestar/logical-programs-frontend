What: In D3.js, Selections & Data Binding are the foundation for creating dynamic visualizations. A selection lets you target DOM elements, and data binding connects your dataset to those elements.

Why: They enable data-driven rendering, meaning shapes, text, and styles update automatically based on your dataset. This is what makes D3 powerful for interactive charts and real-time visualizations.

Where: Use selections and data binding whenever you need to map data to visuals, such as generating bars in a bar chart, circles in a scatter plot, or labels in a graph.

Selections & Data Binding in D3.js


// Example: Binding Data to Paragraphs
constconst data = 'This example demonstrates basic data binding in D3.js using paragraphs';

d3.select("#chart")
  .selectAll("p")      // selects all <p> inside #chart
  .data([data])             // bind data
  .enter()                // handle extra data
  .append("p")       // create new <p> tags
  .text(d => `Value: ${d}`); // use data in text

  
  Note
  This example demonstrates basic data binding in D3.js using paragraphs.

Enter, Update, Exit Example


const numbers = [5, 10, 15];

const selection = d3.select("#update-example")
  .selectAll("div")
  .data(numbers);

// ENTER (new elements)
selection.enter()
  .append("div")
  .text(d => d);

// UPDATE (existing elements)
selection.text(d => d);

// EXIT (remove unused)
selection.exit().remove();
  
  Note
  This example demonstrates the enter-update-exit pattern in D3.js.
   <div id="update-example" class="practices"></div>

Binding Data to SVG Circles


const dataset = [30, 50, 70, 90];

d3.select("#circle-chart")
  .selectAll("circle")
  .data(dataset)
  .enter()
  .append("circle")
  .attr("cx", (d, i) => (i * 70) + 50)
  .attr("cy", 100)
  .attr("r", d => d / 5)
  .attr("fill", "lightblue")
  .attr("stroke", "black");
