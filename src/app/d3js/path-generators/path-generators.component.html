What: Path Generators in D3 create SVG path data (the "d" attribute) from data — e.g. lines, areas, arcs, and links.

Why: Path generators let you declaratively convert arrays and geometry into smooth, parameterized shapes. They reduce manual path math and make transitions, curves, and shapes easier to build.

Where: Use path generators for line charts, area charts, pies/donuts, node-link diagrams, radial charts, icons (symbols), and any custom SVG shape driven by data.

D3 Path Generators — Step-by-step tutorial & examples

Quick setup: include D3 (v6/7) in your page or import it into Angular/TypeScript: import * as d3 (or import * as d3 in TS).

1. Line generator (d3.line)

Use d3.line() to map data points to an SVG path. You can set x/y accessors and curve interpolations.

// Data (x,y pairs)
const data = [
  {x: 0, y: 30},
  {x: 50, y: 80},
  {x: 100, y: 45},
  {x: 150, y: 120},
  {x: 200, y: 70}
];

// scales
const x = d3.scaleLinear().domain([0, 200]).range([10, 390]);
const y = d3.scaleLinear().domain([0, 140]).range([140, 10]);

// line generator
const lineGen = d3.line <{ x: number; y: number }>()
  .x(d => x(d.x))
  .y(d => y(d.y))
  .curve(d3.curveMonotoneX); // try curveBasis, curveLinear, curveCatmullRom

const svg = d3.select('#line-demo');
svg.selectAll('*').remove();

svg.append('path')
  .datum(data)
  .attr('d', lineGen as any)
  .attr('fill', 'none')
  .attr('stroke', '#2b8cbe')
  .attr('stroke-width', 3);

// Add points
svg.selectAll('circle')
  .data(data)
  .enter()
  .append('circle')
  .attr('cx', d => x(d.x))
  .attr('cy', d => y(d.y))
  .attr('r', 3)
  .attr('fill', '#2b8cbe');

2. Area generator (d3.area)

Create filled shapes between two y-values. Useful for highlighting volume or stacked areas.


// area generator example
const data: Point[] = [
  { x: 0, y: 30 },
  { x: 50, y: 80 },
  { x: 100, y: 45 },
  { x: 150, y: 120 },
  { x: 200, y: 70 }
];

// scales
const x = d3.scaleLinear().domain([0, 200]).range([10, 390]);
const y = d3.scaleLinear().domain([0, 140]).range([140, 10]);

// area generator
const areaGen = d3.area<Point>()
  .x(d => x(d.x))
  .y0(y(0)) // baseline
  .y1(d => y(d.y))
  .curve(d3.curveCardinal);

const svg2 = d3.select('#area-demo');
svg2.selectAll('*').remove();

svg2.append('path')
  .datum(data)
  .attr('d', areaGen)
  .attr('fill', '#a6bddb')
  .attr('stroke', '#3690c0');

3. Arc generator (d3.arc) — pie / donut slices

Use the arc generator to build pie and donut slices; combine with d3.pie() to compute angles from data.

const pieData = [10, 25, 40, 15];
const pie = d3.pie();
const arcs = pie(pieData);

const arcGen = d3.arc()
  .innerRadius(30)   // donut hole
  .outerRadius(70);

const svg3 = d3.select('#arc-demo');
svg3.selectAll('*').remove();

const g = svg3.append('g').attr('transform', 'translate(90,90)');

g.selectAll('path')
  .data(arcs)
  .enter()
  .append('path')
  .attr('d', arcGen as any)
  .attr('fill', (d,i) => d3.schemeCategory10[i])
  .attr('stroke', 'white');

4. Symbol generator (d3.symbol) — icons & markers

Generate marker paths (circle, cross, triangle, star) useful for scatterplots and legends.

const symbols = [d3.symbolCircle, d3.symbolCross, d3.symbolTriangle, d3.symbolStar];
const svg4 = d3.select('#symbol-demo');
svg4.selectAll('*').remove();

const g4 = svg4.append('g').attr('transform', 'translate(60,40)');

symbols.forEach((s, i) => {
  g4.append('path')
    .attr('d', d3.symbol().type(s).size(200)())
    .attr('transform', `translate(${i*40},0)`)
    .attr('fill', d3.schemeTableau10[i]);
});

5. Link generators (d3.linkHorizontal / linkVertical / linkRadial)

Great for drawing edges in trees or node-link diagrams. You supply source/target coordinates and the generator returns a path.

const linkData = [{
  source: {x: 10, y: 20},
  target: {x: 150, y: 100}
}];

const linkGen = d3.linkHorizontal< any, any > ()
  .x(d => d.x)
  .y(d => d.y);

const svg5 = d3.select('#link-demo');
svg5.selectAll('*').remove();

svg5.append('path')
  .datum(linkData[0])
  .attr('d', linkGen)
  .attr('stroke', '#666')
  .attr('fill', 'none')
  .attr('stroke-width', 2);

// source/target markers
svg5.selectAll('circle')
  .data([linkData[0].source, linkData[0].target])
  .enter()
  .append('circle')
  .attr('cx', d => d.x)
  .attr('cy', d => d.y)
  .attr('r', 3)
  .attr('fill', '#666');
