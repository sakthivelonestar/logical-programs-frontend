‚ö° Step 3 ‚Äî Custom Directive

Highlight Directive (highlight.directive.ts)


import { Directive, ElementRef, HostListener } from '@angular/core';

@Directive({
  selector: '[appHighlight]'
})
export class HighlightDirective {
  constructor(private el: ElementRef) {}

  @HostListener('mouseenter') onMouseEnter() {
    this.el.nativeElement.style.backgroundColor = 'yellow';
  }

  @HostListener('mouseleave') onMouseLeave() {
    this.el.nativeElement.style.backgroundColor = '';
  }
}

Usage:


<p appHighlight>Hover over me to highlight!</p>

15 Important Interview Questions on Directives

  1. What is a directive in Angular?

    A directive is a class that adds behavior to elements in Angular applications.

  2. What are the types of directives in Angular?

    Three types: Components (with templates), Structural directives (change DOM structure), Attribute directives (change appearance/behavior).

  3. What is the difference between Structural and Attribute directives?

    Structural directives modify DOM layout (e.g., *ngIf, *ngFor), while attribute directives modify element style/behavior (e.g., [ngClass], [ngStyle]).

  4. What does *ngIf do?

    It conditionally includes or removes elements from the DOM based on a Boolean expression.

  5. What is *ngFor used for?

    It iterates over a collection and creates a DOM element for each item.

  6. What is *ngSwitch?

    A structural directive that conditionally renders elements based on matching cases.

  7. What does ngClass do?

    It adds or removes CSS classes dynamically based on expressions.

  8. What does ngStyle do?

    It sets inline styles dynamically on an element based on component data.

  9. Can you create a custom directive?

    Yes, using the @Directive decorator with custom logic.

  10. What is the difference between a Component and a Directive?

    A Component is a directive with a template. Directives without templates are structural or attribute directives.

  11. What is HostListener in a directive?

    It listens to DOM events on the host element and triggers directive methods.

  12. What is HostBinding in a directive?

    It binds a property in the directive to a property of the host element.

  13. Can we use multiple directives on a single element?

    Yes, Angular allows applying multiple directives to the same element.

  14. What is the purpose of the selector in a directive?

    The selector defines how the directive is applied (attribute, element, or class).

  15. Give a use case for custom directives.

    Custom directives can be used for tooltips, hover effects, auto-focus, or reusable UI behaviors.
