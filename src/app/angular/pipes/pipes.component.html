üõ† Step 2 ‚Äî Custom Pipe

Truncate Pipe (truncate.pipe.ts)


import { Pipe, PipeTransform } from '@angular/core';

@Pipe({
  name: 'truncate'
})
export class TruncatePipe implements PipeTransform {
  transform(value: string, limit: number = 20): string {
    return value.length > limit ? value.substring(0, limit) + '...' : value;
  }
}

Usage:


<p>{{ 'This is a very long sentence for demo.' | truncate:15 }}</p>

üëâ Custom pipes let you implement reusable transformations not covered by built-in ones.

15 Important Interview Questions on Pipes

  1. What is a pipe in Angular?

    A pipe is a function that transforms input data into a desired output format in templates.

  2. Give examples of built-in pipes.

    Examples include uppercase, lowercase, date, currency, json, and percent.

  3. How do you create a custom pipe?

    Use the @Pipe decorator and implement the PipeTransform interface with a transform() method.

  4. What is the difference between pure and impure pipes?

    Pure pipes run only when input values change, while impure pipes run on every change detection cycle.

  5. What is the default behavior of custom pipes?

    By default, custom pipes are pure (optimized for performance).

  6. Can pipes take arguments?

    Yes, arguments can be passed after a colon, e.g., { { amount | currency:'EUR' }}.

  7. Can we chain multiple pipes?

    Yes, Angular allows chaining pipes like { { today | date:'short' | uppercase }}.

  8. What is the difference between a pipe and a function call in template?

    Pipes are optimized and cached, while template functions may cause performance issues since they run on every change detection cycle.

  9. What is the async pipe?

    The async pipe automatically subscribes to Observables/Promises and displays their latest value.

  10. What is a parameterized pipe?

    A pipe that accepts arguments to control its transformation, e.g., { { text | slice:0:10 }}.

  11. Can we use pipes in component TypeScript code?

    Pipes are designed for templates, but you can inject and call them in code if needed.

  12. What is an impure custom pipe use case?

    Useful for filtering arrays or objects that change frequently during runtime.

  13. What is the difference between slice pipe and ngFor with slice?

    The slice pipe transforms displayed output, while *ngFor with slice limits rendered items in the DOM.

  14. How does Angular optimize pure pipes?

    Pure pipes execute only when inputs change, reducing unnecessary recalculations.

    A truncate pipe to shorten long text with "..." at the end.
