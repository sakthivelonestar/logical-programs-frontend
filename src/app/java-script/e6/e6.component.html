JavaScript DOM Manipulation

What is DOM Manipulation?

The DOM (Document Object Model) represents the HTML structure of a webpage. JavaScript can manipulate DOM elements to dynamically change content, style, and structure.
  • Access HTML elements using selectors
  • Modify content, attributes, and styles
  • React to user interactions (events)
  • Create, remove, or move elements

Selecting Elements

  • getElementById – select by ID
  • getElementsByClassName – select by class
  • getElementsByTagName – select by tag
  • querySelector / querySelectorAll – CSS-style selectors
// By ID
const header = document.getElementById('main-header');

// By Class
const items = document.getElementsByClassName('list-item');

// By Tag
const paragraphs = document.getElementsByTagName('p');

// CSS selectors
const firstItem = document.querySelector('.list-item:first-child');
const allItems = document.querySelectorAll('.list-item');

Changing Content & Attributes

  • Use textContent or innerHTML to change content
  • Use setAttribute and getAttribute to modify attributes
  • Modify styles using style property
// Change text
header.textContent = 'Hello World';
header.innerHTML = '<span>Hello World</span>';

// Change attributes
header.setAttribute('class', 'new-header');
console.log(header.getAttribute('class'));

// Change styles
header.style.color = 'blue';
header.style.fontSize = '24px';

Creating & Removing Elements

  • Create elements with createElement
  • Add them to DOM with appendChild or insertBefore
  • Remove elements with removeChild or remove()
// Create a new paragraph
const para = document.createElement('p');
para.textContent = 'I am new!';

// Append to body
document.body.appendChild(para);

// Remove element
document.body.removeChild(para); // or para.remove();

Event Handling

  • Attach events using addEventListener
  • Remove events using removeEventListener
  • Use event object to access event properties
const button = document.querySelector('button');

button.addEventListener('click', function(event) {
  console.log('Button clicked!');
  console.log(event.target); // element that triggered the event
});

Key Points

  • DOM Manipulation allows dynamic control of HTML, CSS, and user interactions
  • Always select elements before manipulating
  • Use event listeners instead of inline events for cleaner code
  • Remember performance: minimize DOM updates inside loops
