What: In JavaScript, Date & Time Concepts allow you to work with dates, times, and timestamps, including creating, formatting, and performing calculations on date objects.

Why: They help you manage time-based data, schedule events, calculate differences between dates, and format timestamps for display or storage in applications.

Where: You can use them in web applications, scheduling apps, logs, calendars, timers, or any feature that relies on date and time management.

1. Creating Date Objects

Learn how to create and initialize JavaScript Date objects:

Examples

// Current date and time
const now = new Date();
console.log(now);

// Specific date
const birthday = new Date("1990-05-15");
console.log(birthday);

// Date with year, month, day, hours, minutes, seconds
const customDate = new Date(2025, 8, 20, 14, 30, 0);
console.log(customDate);

Key Points

  • Months are zero-indexed (January = 0, December = 11).
  • Day and time components default to 0 if not provided.
  • String parsing may depend on browser implementation.

2. Date Methods

Methods to extract and manipulate date components:

Getting Date Components

// Example Date
const today = new Date();

console.log(today.getFullYear());  // Year
console.log(today.getMonth());     // Month (0-11)
console.log(today.getDate());      // Day of the month
console.log(today.getDay());       // Day of the week (0-6)
console.log(today.getHours());     // Hours (0-23)
console.log(today.getMinutes());   // Minutes
console.log(today.getSeconds());   // Seconds

Setting Date Components

// Update specific parts of a date
today.setFullYear(2026);
today.setMonth(0);      // January
today.setDate(1);
today.setHours(12);
console.log(today);

3. Date Calculations

Perform calculations such as differences between dates and adding days:

Difference Between Dates

const start = new Date("2025-09-01");
const end = new Date("2025-09-20");

const diffTime = end - start; // Difference in milliseconds
const diffDays = diffTime / (1000 * 60 * 60 * 24);
console.log(diffDays); // 19 days

Adding Days to a Date

function addDays(date, days) {
  const result = new Date(date);
  result.setDate(result.getDate() + days);
  return result;
}

console.log(addDays(new Date(), 10)); // 10 days later

4. Formatting Dates

Display dates in readable formats:

Common Formatting Methods

const now = new Date();

console.log(now.toDateString());     // e.g., "Sat Sep 20 2025"
console.log(now.toTimeString());     // e.g., "14:30:00 GMT+0530"
console.log(now.toLocaleDateString()); // e.g., "20/09/2025"
console.log(now.toISOString());        // e.g., "2025-09-20T08:00:00.000Z"

5. Timers & Intervals

Use Date objects with setTimeout and setInterval for scheduling:

Examples

// Execute after a delay
setTimeout(() => {
  console.log("Executed after 2 seconds");
}, 2000);

// Execute repeatedly every 1 second
const timer = setInterval(() => {
  console.log(new Date().toLocaleTimeString());
}, 1000);

// Stop interval after 5 seconds
setTimeout(() => clearInterval(timer), 5000);

6. Key Points

  • JavaScript Date object represents a single point in time.
  • Months are zero-indexed; days and times start at 0.
  • Millisecond timestamps are used for date calculations.
  • Use formatting methods to display dates for different locales.
  • Timers allow scheduling tasks using setTimeout and setInterval.
