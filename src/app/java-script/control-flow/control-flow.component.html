What: Control flow in JavaScript is the way the program decides which code to run next based on conditions, loops, or function calls.

Why: It is important because it allows the program to make decisions, repeat tasks, and handle errors, making the code dynamic and intelligent.

Where: Control flow is used in if-else statements, loops (for, while), functions, switch cases, and error handling to manage how and when parts of the code are executed.
