Swagger Documentation in NestJS

Step-by-step guide for integrating Swagger in your NestJS project

Steps to Implement Swagger

Step 1: Installation

npm install --save @nestjs/swagger swagger-ui-express
Copied!

Step 2: Global Prefix Setup

// main.ts
const globalPrefix = 'api';
app.setGlobalPrefix(globalPrefix);
Copied!

Step 3: Swagger Configuration

// main.ts
import { DocumentBuilder, SwaggerModule } from '@nestjs/swagger';

const config = new DocumentBuilder()
  .setTitle('My API')
  .setDescription('API documentation for my NestJS project')
  .setVersion('1.0')
  .addTag('users')
  .build();

const document = SwaggerModule.createDocument(app, config);
SwaggerModule.setup('api-docs', app, document);
Copied! {{tooltip3|json}}

Step 4: Access Documentation

http://localhost:3000/api-docs
Copied!
