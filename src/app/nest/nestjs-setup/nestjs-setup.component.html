What: NestJS is a progressive Node.js framework that uses TypeScript by default and follows modular architecture patterns inspired by Angular.

Why: It provides structure, scalability, and maintainability to Node.js applications through decorators, dependency injection, and organized project structure.

Where: Used for building enterprise-grade APIs, microservices, GraphQL applications, and scalable backend systems in companies of all sizes.

Step 1: Installation & Project Creation

-- 🚀 Install NestJS CLI globally
npm i -g @nestjs/cli

-- 📁 Create a new NestJS project
nest new my-nestjs-app

-- 🏃‍♂️ Navigate to project and start development server
cd my-nestjs-app
npm run start:dev

Getting started with NestJS installation and creating your first project.

Step 1.1: Creating Modules and Resources

-- 📦 Generate a Basic Module
# Create a new module (basic structure)
nest generate module users
# or shorter syntax
nest g mo users

-- 🎯 Generate a Complete Resource (CRUD)
# Create a full resource with CRUD operations
nest generate resource products
# or shorter syntax
nest g res products

-- 💡 Resource Generation Options:
